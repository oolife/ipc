# Server

## How to build
bash > cd server <br>
bash > make

## How to run
bash > ./server



# Client

## How to build
bash > cd client<br>
bash > make

## How to run
bash > ./client $(input_file)<br>
Two input files are provided.
Using this input file, you can execute as follows.<br>

bash > ./client ./input_1.txt<br>
or<br>
bash > ./client ./input_2.txt<br>

 You can run multiple clients.
